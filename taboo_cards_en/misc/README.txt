# English taboo-style cards

Files:
- manifest.json
- all_cards_flat.json
- all_cards_grouped.json
- one JSON file per category

Category file structure:
{
  "schema_version": 1,
  "category": "Technology & Internet",
  "language": "en",
  "cards": [
    {
      "id": "technology_and_internet-001",
      "target": "Laptop",
      "taboo": ["computer", "screen", "keyboard", "portable", "battery"]
    }
  ]
}
